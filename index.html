Kia ora!

Welcome to this personal project of mine. My name is Stephen, and I like to code. Today, I am working towards a Software Engineering degree and with some luck a long and successful career in this great field.

Somehow after programming for almost a decade, I've haven't touched HTML, CSS, or done any sort of web design. I was busy spending time with C#, Python, Java, and more recently C, and SQL. But the web is only becoming more exceedingly important than it already is, and learning JavaScript has always been on my to-do list so here we are.

This site is a way for me to wrap my head around the fundamentals of web development, front-end at least, and my current plan is to turn it into a portfolio of sorts. Thank you for visiting my site, and joining me on this exciting journey.
