Kia ora!

Welcome to this personal project of mine. My name is Stephen, and I like to code. Today, I am working towards a Software Engineering degree and with some luck a long and successful career in this great field.

Late 2023, after programming for almost a decade, I hadn't touched HTML, CSS, or done any sort of web design. I was getting busy with C#, Python, Java, & C. But as each day passes, the web only becomes more exceedingly important, so I thought why not make a start with a portfolio site?

So I built this static site from scratch as a way for me learn the fundamentals of HTML and CSS. Thank you for visiting my site, and joining me on this exciting journey. Have a good one!
